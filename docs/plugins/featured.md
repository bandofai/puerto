## 1. Essential Infrastructure

**Start here** - Core tools every user should install first.

### essentials
**The foundation for everything**

Provides 4 powerful MCP servers and comprehensive requirements management.

**Includes:**
- **Serena** - Semantic code navigation
- **Context7** - Up-to-date documentation lookup
- **Sequential Thinking** - Structured problem-solving
- **Playwright** - Browser automation
- **Commands:** `/brainstorm`, `/implement`, `/req-list`, `/req-update`, `/req-tests`, `/req-status`, `/continue`

```bash
/plugin install essentials@puerto
```

**Why it matters:** This is your development workflow foundation - semantic code understanding, real-time docs, structured thinking, and browser testing in one plugin.
