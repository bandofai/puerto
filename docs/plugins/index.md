## What's Available

### 7 Department Plugins
Each department plugin contains multiple specialized agents working together:

- **🔧 Engineering** (7 agents) - Full-stack development
- **🎨 Design** (4 agents) - UX research, writing, accessibility
- **📢 Marketing** (5 agents) - Content, SEO, social, growth
- **📊 Product** (4 agents) - Product management, analytics
- **💼 Sales** (3 agents) - Sales, customer success, partnerships
- **⚙️ Operations** (3 agents) - HR, operations, office management
- **🎯 Leadership** (1 agent) - Strategic planning

### 7 Infrastructure Tools
Essential tools and MCP servers:

- **essentials** - Core MCP servers + requirements management
- **chrome-devtools** - Browser automation
- **figma** - Design-to-code generation
- **subagent-creator** - Agent architecture specialist
- **claude-md-master** - CLAUDE.md configuration
- **cloudflare** - Cloudflare Pages deployment
- **orchestrator** - Multi-agent workflow coordination

## Quick Install

```bash
# Add marketplace
/plugin marketplace add bandofai/puerto

# Essential infrastructure
/plugin install essentials@puerto

# Department teams
/plugin install engineering@puerto
/plugin install design@puerto
/plugin install marketing@puerto
/plugin install product@puerto
/plugin install sales@puerto
/plugin install operations@puerto
/plugin install leadership@puerto
```
