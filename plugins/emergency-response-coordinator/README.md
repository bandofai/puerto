# Emergency Response Coordinator Plugin

Emergency situation coordination specialist for activating protocols, assessing situations, notifying stakeholders, and coordinating resources during crises.

## Agents

- **incident-assessor** (Sonnet): Rapid situation assessment with severity scoring (P1-P4)
- **protocol-activator** (Haiku, Fast): Emergency protocol activation from playbooks
- **stakeholder-notifier** (Haiku, Skill-Aware): Stakeholder notifications and status updates
- **resource-coordinator** (Sonnet): Resource allocation and team coordination

## Skills

- emergency-protocols: Incident classification, response playbooks, escalation paths
- stakeholder-communication: Notification templates, status updates, crisis communication
- incident-management: ICS framework, NIMS guidelines, post-incident reviews

## Features

✓ Rapid incident assessment (< 5 minutes)
✓ Severity scoring (P1: Critical to P4: Low)
✓ Protocol activation from playbooks
✓ Multi-channel stakeholder notification
✓ Resource coordination
✓ Incident command system (ICS) alignment
✓ Post-incident reporting
