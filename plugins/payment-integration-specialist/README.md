# Payment Integration Specialist Plugin

Payment integration specialist for payment gateway integration, PCI DSS compliance, fraud prevention, recurring billing setup, and payment reconciliation.

## Overview

The Payment Integration Specialist plugin provides agents for secure payment processing including gateway integration (Stripe, PayPal, Braintree), PCI DSS compliance validation, fraud detection and prevention, and payment reconciliation.

## Agents

### 1. gateway-integrator (Sonnet, Skill-Aware)
Integrates payment gateways with proper error handling and security.

**Use for**: Gateway integration, API implementation, webhook handling

### 2. compliance-specialist (Sonnet, Skill-Aware)
Ensures PCI DSS compliance with security validation and audit preparation.

**Use for**: PCI compliance, security audits, compliance documentation

### 3. fraud-preventer (Sonnet, Skill-Aware)
Implements fraud detection rules and risk scoring systems.

**Use for**: Fraud detection, risk assessment, chargeback prevention

### 4. reconciliation-manager (Haiku, Fast)
Reconciles payment transactions with accounting systems.

**Use for**: Transaction reconciliation, reporting, dispute management
