# Design Department

**User-centered design team with 4 specialized roles covering UX research, writing, accessibility, and creative direction.**

---

## Team Structure

### Leadership
- **Design Lead** - Creative direction, brand strategy, design systems

### Specialists
- **UX Researcher** - User research, usability testing, insights
- **UX Writer** - Content strategy, microcopy, voice and tone
- **Accessibility Specialist** - WCAG compliance, inclusive design

---

## When to Use This Department

**Strategy & Planning:**
- Design Lead: Creative direction and brand strategy

**Research & Insights:**
- UX Researcher: User research and usability testing

**Content & Copy:**
- UX Writer: User-facing text and content strategy

**Accessibility:**
- Accessibility Specialist: WCAG compliance and inclusive design

---

## Skills Coverage

**UX Research**
- User interviews and surveys
- Usability testing
- Persona development
- Journey mapping
- Analytics analysis
- Research synthesis

**UX Writing**
- Microcopy and content strategy
- Voice and tone guidelines
- Error messages
- Onboarding copy
- Information architecture

**Accessibility**
- WCAG 2.1 compliance
- Screen reader compatibility
- Keyboard navigation
- ARIA implementation
- Inclusive design
- Accessibility testing

**Brand Strategy**
- Visual identity
- Brand guidelines
- Design systems
- Creative direction

## Usage Examples

**Feature Design:**
```
1. UX Researcher: Conduct user research
2. Design Lead: Define creative direction
3. UX Writer: Create content strategy
4. Accessibility Specialist: Ensure WCAG compliance
```

**Design System:**
```
1. Design Lead: Architect design system
2. UX Writer: Define content patterns
3. Accessibility Specialist: Create accessible components
4. UX Researcher: Validate with users
```
