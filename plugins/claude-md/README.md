# Claude.md

Create effective CLAUDE.md files that transform Claude Code from a general-purpose assistant into a project-specialized tool.

## Installation

```bash
/plugin install claude-md@puerto
```

## Features

- **CLAUDE.md creation** - Analyze codebases and generate optimized project instructions
- **Quality auditing** - Identify anti-patterns and fix token-wasting content
- **Monorepo strategy** - Hierarchical CLAUDE.md structure for large codebases
- **Writing patterns** - Concise, actionable documentation techniques

## Usage

### Create CLAUDE.md

```
"Create a CLAUDE.md for this project"
"Set up Claude Code configuration"
```

### Audit Existing

```
"Audit my CLAUDE.md for quality"
"Optimize my project documentation for Claude"
```

### Monorepo Setup

```
"Set up CLAUDE.md hierarchy for this monorepo"
"Optimize agent context for workspace packages"
```

## Key Concepts

| Concept | Description |
|---------|-------------|
| Golden Rule | Document what Claude gets wrong, not everything |
| Token Budget | Keep <500 lines, <15k tokens total |
| Hierarchy | Root + subdirectory files for monorepos |
| Iteration | Start minimal, expand based on friction |
