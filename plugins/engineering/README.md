# Engineering Department

**Full-stack engineering team with 7 specialized roles covering frontend, backend, mobile, data, DevOps, and QA.**

---

## Team Structure

### Leadership
- **Engineering Lead** - System architecture, technical strategy, team coordination

### Specialists
- **Frontend Engineer** - UI development, React/Vue/Svelte, state management, accessibility
- **Backend Engineer** - APIs, databases, authentication, business logic
- **DevOps Engineer** - CI/CD, infrastructure, Kubernetes, monitoring
- **Mobile Engineer** - iOS/Android, React Native/Flutter, mobile UI/UX
- **Data Engineer** - Data pipelines, ETL, data warehousing, ML infrastructure
- **QA Engineer** - Testing strategy, automation, quality assurance

---

## When to Use This Department

**Discovery & Planning:**
- Engineering Lead: System architecture and technical decisions

**Implementation:**
- Frontend Engineer: Building user interfaces
- Backend Engineer: APIs and business logic
- Mobile Engineer: Mobile applications
- Data Engineer: Data pipelines and analytics infrastructure

**Infrastructure & Quality:**
- DevOps Engineer: Deployment and monitoring
- QA Engineer: Testing and quality assurance

## Usage Examples

**Full-Stack Feature Development:**
```
1. Engineering Lead: Design system architecture
2. Backend Engineer: Build API endpoints
3. Frontend Engineer: Create UI components
4. QA Engineer: Write automated tests
5. DevOps Engineer: Set up CI/CD and deploy
```

**Mobile App Development:**
```
1. Engineering Lead: Define mobile architecture
2. Mobile Engineer: Build iOS/Android app
3. Backend Engineer: Create mobile-specific APIs
4. QA Engineer: Mobile testing strategy
5. DevOps Engineer: App distribution and monitoring
```

**Data Infrastructure:**
```
1. Engineering Lead: Design data architecture
2. Data Engineer: Build ETL pipelines
3. Backend Engineer: Create data access APIs
4. DevOps Engineer: Deploy data infrastructure
5. QA Engineer: Data quality validation
```
