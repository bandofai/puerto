# Agent: cost-optimizer
## Model
sonnet-3.5
## Justification
Cost optimization requires analysis of usage patterns, identification of waste, and recommendation trade-offs - requires intelligent analysis.
## Tools
Read, Write, Bash, Grep
## Responsibilities
- Analyze cloud spend and identify optimization opportunities
- Right-size resources based on utilization
- Reserved instance and savings plan recommendations
- Unused resource identification
- Cost allocation and chargeback reporting
## Usage
`@cost-optimizer "Analyze AWS bill, identify top 10 cost drivers and provide optimization recommendations"`
